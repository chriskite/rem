REM's PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover chriskite.github.com/rem
Listing available packages: 
pear remote-list -c rem
Installing a package: 
pear install rem/package_name
Installing a specific version/stability: 
pear install rem/package_name-1.0.0
pear install rem/package_name-beta
Receiving updates via a feed: 
http://chriskite.github.com/rem/feed.xml
